# AgroGestión 🌱

Sistema de gestión agrícola desarrollado con **Flask y MySQL**.

## Funcionalidades actuales

- Login de usuarios
- Gestión de usuarios (CRUD)
- Gestión de cultivos
- Base de datos relacional para gestión agrícola

## Tecnologías utilizadas

- Python
- Flask
- MySQL
- HTML / Jinja2
